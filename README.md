# Customer Churn Analysis — Telco Subscription Business

Analysis of why a subscription telecom company is losing 26.5% of its
customers annually, and what specifically drives that churn.

## What's in this repo
- `churn_analysis.py` — Python (pandas) data cleaning and exploration
- `churn_analysis_queries.sql` — the same analysis written in SQL
- `telco_churn_cleaned.csv` — cleaned dataset, ready for BI tools

## Key findings
- Month-to-month customers churn at 42.7% vs. just 2.8% for two-year contracts
- New customers churn at 52.9% in their first 6 months
- Customers without tech support churn at 41.6% vs. 15.2% with support

## Tools used
Python, pandas, SQL, Power BI
